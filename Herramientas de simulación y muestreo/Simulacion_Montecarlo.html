1 Estimar \(\pi\) mediante el mtodo de Monte Carlo a travs del ejemplo (explicado en clase) de las gotas de lluvia que caen dentro del crculo tras realizar el experimento \(N = 10^4\) veces. Dar el intervalo de confianza con un nivel de confianza del 97 %. Analizar los resultados.

El mtodo de las gotas de lluvia consiste en considerar un crculo de radio unidad circunscrito en un cuadrado y generar puntos y ver cuantos han caido dentro del crculo y cuanto fuera. Estos puntos vienen de la generacin aleatoria de una variable uniforme entre \(-1\) y \(1\).

N = 10^4
accept = c()

set.seed(123)
sample.x <- runif(N,-1,1)
set.seed(1234)
sample.y <- runif(N,-1,1)

Consideramos ahora la funcin de la circunferencia y vemos cuales de los puntos generados han caido dentro del circulo.

F = function(x,y){x^2+y^2}

for(i in 1:length(sample.x)){
  if(F(sample.x[i],sample.y[i]) <= 1){
    accept[i] = 1
  } 
  else if(F(sample.x[i],sample.y[i]) > 1){
    accept[i] = 0
  }
}

Para calcular la estimacin de \(\pi\), hemos de utilizar la siguiente formula.

\[ \hat{\pi} = 4*\dfrac{puntos\_dentro\_del\_circulo}{puntos\_totales\_generados} \]

Zhat = 4*sum(accept)/N

phat = sum(accept)/N

Vemos que la estimacin de \(\pi\) que obtenemos mediante el mtodo de las gotas de lluvia es de 3.134 y la probabilidad estimada es de 0.7835. Pasemos a observar cul es el intervalo de confianza al \(97\)%.

## Intervalo 97%

x = qnorm(1-(1-0.97)/2)

CI = c(phat - x*sqrt(phat*(1-phat)/N),phat + x*sqrt(phat*(1-phat)/N))
CI
## [1] 0.7745623 0.7924377

Vemos que el intervalo de confianza al \(97\)% para la probabilidad es de (0.7745623, 0.7924377), por lo tanto, el intervalo de confianza para \(\pi\) sera (3.0982492, 3.1697508).

2 Simlese una variable aleatoria X ~ Beta(3, 6) mediante otra variable beta Y ~ Beta(2.7, 5.6) como envolvente.

Lo que haremos para resolver este ejercicio ser, primero, generar \(10^6\) muestras de una variable Beta(3, 6).

x = seq(0,1,by=0.001)
set.seed(123)
sample.x = rbeta(10^6,2.7,5.6)

Ahora necesitamos calcular la cota de la funcin \(f(x)/g(x)\) donde \(f(x)\) es la funcin de distribucin de una variable Beta(3, 6) y \(g(x)\) es la funcin de distribucin de una variable Beta(2.7, 5.6) que ser nuestra envolvente. Esta cota la hemos calculado mediante wolframalpha y hemos obtenido que es de \(1.067\). Pasamos a aplicar el mtodo general de aceptacin-rechazo.

accept = c()

for(i in 1:length(sample.x)){
  U = runif(1, 0, 1)
  if(dbeta(sample.x[i], 2.7, 5.6)*(1.067)*U <= dbeta(sample.x[i], 3, 6)) { 
    accept[i] = 'Yes' 
  } 
  else if (dbeta(sample.x[i], 2.7, 5.6)*(1.067)*U > dbeta(sample.x[i], 3, 6)){
    accept[i] = 'No'
  }
}

3 Mediante el mtodo Hit and Miss determnese el valor de la siguiente integral:

\[ \int_{0}^{1} (\cos(2x)+ \sin(4x))^2 dx \]

Para utilizar el mtodo Hit and Miss, hemos de ver que nuestra integral se encuentra en un intervalo acotado (que lo est ya que se encuentra entre 0 y 1), hemos de ver que la funcin que se integra es positiva en el dominio de integracin (que lo es ya que est elevada al cuadrado) y hemos de calcular el mximo de la funcin \((\cos(2x)+ \sin(4x))^2\) en ese intervalo. Utilizando la pagina web de wolframalpha, se ha obtenido que el valor mximo de la funcin en el intervalo \([0,1]\) es, aproximadamente, de \(3.09821\).

Ahora podemos pasar al mtodo Hit and Miss. Lo primero que haremos ser dibujar nuestra funcin.

x = seq(0,1,by=0.001)
f = function(x){(cos(2*x)+ sin(4*x))^2}
y = f(x)
df <- data.frame(x,y)

ggplot(df, aes(x))+
  geom_line(aes(y=y), colour="red")

Ahora generaremos \(10^6\) valores de una distribucin uniforme \([0,1]\) y vamos generando valores con una distribucin uniforme \([0,3.09821]\) formando pares (x,u) donde x viene de la uniforme \([0,1]\) y u viene de la uniforme \([0,3.09821]\). Entonces vemos cuales de estos pares estan debajo de la funcin que se integra y el valor de la integral sera el siguiente:

\[ \hat{I} = 3.09821*(1-0)*\dfrac{puntos\_debajo\_de\_la\_curva}{puntos\_totales\_generados} \]

sample.x = runif(10^6,0,1)
accept = c()
sample.accept = c()

for(i in 1:length(sample.x)){
  U = runif(1, 0, 1)
  if(dunif(sample.x[i], 0, 1)*(3.09821)*U <= f(sample.x[i])) { 
    accept[i] = 'Yes'
    sample.accept[i] = 1
  } 
  else if(dunif(sample.x[i],0,1)*(3.09821)*U > f(sample.x[i])) {
    accept[i] = 'No'
    sample.accept[i] = 0
  }
}

T = data.frame(sample.x, accept = factor(accept, levels= c('Yes','No')), sample.accept)

phat = sum(sample.accept)/10^6
I = (3.09821)*(1-(0))*phat

El valor estimado de la integral es de 1.5575631. Calculemos ahora el valor real de esta integral.

Ireal = integrate(f,0,1)

Vemos que el valor real de esta integral es de \(1.558277\) con un error absoluto menor a \(1.7*10^{-14}\). Por ultimo, calculemos el error cometido con la aproximacin.

SE = (3.09821)*sqrt(phat*(1-phat))/sqrt(10^6)

Vemos que el error es de 0.0015491, que es un error muy pequeo.

4.2 Obtener mediante alguna tcnica de simulacin de Monte Carlo el valor de la siguiente integral:

\[ \int_{\pi/2}^{3\pi/2} \cos(x) dx \]

Lo que haremos ser resolver esta ecuacin por el mtodo general de integracin Monte Carlo utilizando la funcin uniforme, es decir:

\[ \int_{\pi/2}^{3\pi/2} \cos(x) dx = \int_{\pi/2}^{3\pi/2} (3\pi/2-\pi/2)\cos(x)f_U(x) dx \]

siendo \(G(x)=(3\pi/2-\pi/2)\cos(x)\) y \(f_U(x)\) es la funcin de densidad de la uniforme en \([\pi/2,3\pi/2]\). Lo que hacemos ahora es simular \(10^6\) muestras de la uniforme \(U[\pi/2,3\pi/2]\) y la aproximacin del valor de la integral se calcula de la siguiente forma:

\[ G(U)_{10^6}= \sum^{10^6}_{i=1} \dfrac{G(U_i)}{N} \] Utilizaremos la siguiente funcin para el clculo de la integral

integral <- function(N,a,b,f){
  set.seed(123)
  X <- runif(N, a, b)
  Y <- (b-a)*f(X)
  Int <- sum(Y)/N
  return(Int)
}

Pasemos a calcular la integral mediante una aproximacin y su valor real.

g <- function(x){cos(x)}

I <- integral(10^6, pi/2,3*pi/2, g)
Ireal = integrate(g,pi/2,3*pi/2)

Observamos que el valor de la integral aproximandola es -1.9989661 y el valor real de la integral es \(-2\), por lo que podemos decir que la aproximacin es bastante buena.

4.3 Podramos integrar mediante el mtodo de Hit and Miss? Razona la respuesta.

Para ver si podemos integrar mediante el mtodo de Hit and Miss, hemos de ver si se satisfacen las hiptesis. Primero se ha de cumplir que el intervalo de integracin sea acotado que en este caso se cumple ya que el intervalo es \([\pi/2,3\pi/2]\). Lo siguiente que hemos de ver es que esta funcin en el intervalo de integracin es positiva. Esto no ocurre ya que por ejemplo \(\cos(\pi) = -1\). Por lo tanto en un principio no podra aplicarse el mtodo de Hit and Miss.

Para solucionar este problema (ya que al ser una funcin continua en un intervalo acotado, esta tiene un mximo, que es la hiptesis que falt comprobar antes), lo que podemos hacer es integrar la funcin \(-\cos(x)\) en el mismo intervalo. Esta nueva funcin es positiva en todo el intervalo y satisface las dems propiedades, entoces podemos aplicar el mtodo Hit and Miss, lo nico que al resultado de la estimacin habra que cambiarle el signo. Veamoslo.

f <- function(x){-cos(x)}
set.seed(123)
sample.x = runif(10^6,pi/2,3*pi/2)
accept = c()
sample.accept = c()

for(i in 1:length(sample.x)){
  a = runif(1,0,1) 
  if(a <= f(sample.x[i])) { 
    accept[i] = 'Yes'
    sample.accept[i] = 1
  } 
  else if(a  > f(sample.x[i])) {
    accept[i] = 'No'
    sample.accept[i] = 0
  }
}

T = data.frame(sample.x, accept = factor(accept, levels= c('Yes','No')), sample.accept)

phat = sum(sample.accept)/10^6
I = (3*pi/2-(pi/2))*phat

Hemos obtenido una aproximacin de 1.9997525, que si le cambiamos el signo es muy parecido a \(-2\) que es el valor real de la integral.

Otra opcin que podemos hacer para aplicar el mtodo de Hit and Miss sobre la integral del coseno es, ya que la funcin es peridica, cambiar el intervalo de integracin a por ejemplo \([-\pi/2, \pi,2]\) y cambiarle el signo al resultado.
